# Proyecto de Inmobiliaria de Crowdfunding

Este proyecto es una simulación de una plataforma de crowdfunding inmobiliario desarrollada en JSP. La plataforma permite a los usuarios registrarse como inversores o gestores y participar en proyectos inmobiliarios.

## Descripción del Proyecto

La plataforma de crowdfunding inmobiliario permite a los inversores financiar proyectos inmobiliarios y a los gestores administrar dichos proyectos. Los principales componentes del sistema son:

- **Proyectos**: Proyectos inmobiliarios disponibles para inversión.
- **Usuarios Inversores**: Usuarios que pueden invertir en los proyectos.
- **Gestores**: Usuarios que administran y gestionan los proyectos.

## Funcionalidades

- **Registro y Autenticación**: Los usuarios pueden registrarse y autenticarse en la plataforma.
- **Gestión de Proyectos**: Los gestores pueden crear, editar y eliminar proyectos inmobiliarios.
- **Inversión en Proyectos**: Los inversores pueden ver los proyectos disponibles y realizar inversiones.
- **Panel de Usuario**: Los usuarios tienen acceso a un panel donde pueden ver sus inversiones y proyectos gestionados.

## Tecnologías Utilizadas

- **Java**: Lenguaje de programación principal.
- **JSP (JavaServer Pages)**: Para la generación de contenido dinámico en el servidor.
- **Servlets**: Para manejar las solicitudes HTTP.
- **MySQL**: Base de datos para almacenar la información de usuarios y proyectos.
- **HTML/CSS**: Para la estructura y el diseño de la interfaz de usuario.
